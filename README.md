# glencoe-microsat

This work is derived from [microsat](https://github.com/marijnheule/microsat) by Marijn Heule and Armin Biere.

## Build

	./configure && make

## Build and Install

	./configure && sudo make install

## Usage
### Version info

	./microsat --version

### Solving a DIMACS encoded SAT problem
	./microsat DIMACS_FILE
